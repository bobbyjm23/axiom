**Date:** 2026-07-19  
**Status:** Design approved (conversational); awaiting user review of this spec before implementation plan  
**Approach:** Foundation Sprint (organic-first + $500/mo paid learning)

## Goal

Stand up a founder-operable marketing system for **Sovereign Warden** that:

1. Builds company-page LinkedIn authority first  
2. Runs a disciplined **$500/month** paid test (LinkedIn $300 / Google $200)  
3. Switches primary CTA to fit-call bookings after Adneo proof  
4. Gives a complete checklist so nothing critical is missing  

Success in 90 days is a running system + learning, not high demo volume.

---

## Constraints (locked)

| Constraint | Value |
|------------|--------|
| Monthly ad budget | **$500 AUD** ongoing |
| Split | LinkedIn **$300** · Google Search **$200** |
| Primary LinkedIn voice | **Company page first**; founder amplifies |
| CTA sequence | Phase 1 authority/brief → Phase 2 fit-call after Adneo case proof |
| ICP | AU regulated mid-market, 50–250 staff; law #1, accounting #2 |
| Motion | Founder-led; syncs with existing GTM (Adneo-first, law outbound week 7+) |
| Out of scope (90 days) | Full nurture automation, chatbots, Law Society/CPA events (Q3–Q4), budget scale-up without a winner |

## 1. System architecture

Seven pillars, founder-owned, reviewed weekly:

| Pillar | Job | 90-day mode |
|--------|-----|-------------|
| Positioning & offers | One ICP story; CTA changes after Adneo | Soft brief / follow → fit-call |
| LinkedIn Company | Brand home + distribution | Complete profile + 3 posts/week |
| Founder amplification | Reach multiplier | Daily comments/shares; ≤1 personal post/week |
| Website / landing | Convert traffic | sovereignwarden.com + UTMs + tracking |
| Paid ($500/mo) | Cheap learning | Kill losers weekly |
| Outbound sync | Same messages as Priority A sales | No conflicting offers or free law pilots |
| Measurement | Know what to cut | 5 KPIs only |

### CTA phases

1. **Now → Adneo signed + usable proof:** Follow / sovereignty brief / soft contact  
2. **After proof:** Book 30-min fit call as primary CTA everywhere (site, ads, company button)

### 90-day success bar

- Company page complete + consistent posting  
- 2–4 qualified conversations/month from organic + ads combined  
- Clear winner: which LinkedIn audience or Google keyword theme is worth scaling later

## 3. Paid media ($500/mo)

### Shared rules

| Rule | Spec |
|------|------|
| Landing | Phase 1: site or brief capture · Phase 2: fit-call booking (`#contact` / calendar) |
| UTMs | `utm_source=linkedin\|google` · `utm_medium=paid` · `utm_campaign=YYYY-MM-{theme}` |
| Geo | Australia only |
| Offer match | Ad copy must match page H1/CTA |
| Kill rule | Pause any ad set/keyword with **$40+ spend and 0 meaningful actions** |
| Scale rule | Reallocate within $500 only after 2+ weeks of data; no budget increase without a winner |

Meaningful action = brief email, form submit, booking, or qualified DM.

### LinkedIn — $300/mo (~$10/day)

- **Format:** Sponsored Content only (single image or document). No Conversation Ads / InMail at this budget.  
- **Audience A — Law (60% ≈ $180):** Australia; Legal / ops-adjacent functions; Director–CXO–Partner–Owner; titles Managing Partner, Partner, COO, Practice Manager, Risk, GC, IT Manager; company size 51–200 and 201–500.  
- **Audience B — Accounting/advisory (40% ≈ $120):** Same seniority/size; Accounting/Finance/Operations; Managing Partner, Partner, COO, CFO, Risk/Compliance, IT Manager.  
- **Exclusions:** Students, entry-level, companies &lt;11 employees.  
- **Creatives (rotate 2–3):** shadow ChatGPT problem; Copilot cost contrast; Privacy Act education.  
- **CTA:** Learn more / brief (phase 1) → Book discovery call (phase 2).  
- **Refresh:** Creative every 3–4 weeks.

### Google Search — $200/mo (~$6.50/day)

- Search only; Exact + Phrase; Search partners off; Display off.  
- **Themes:** private/on-prem AI for firms; data sovereignty / Privacy Act AI; Copilot alternative / cost.  
- **Negatives (seed):** free, course, job, salary, openai api, chatgpt login, consumer, school, university, template download.  
- **Ads:** 3 RSAs per ad group; sitelinks to site sections / brief / book call (by phase).  
- **Bidding:** Manual CPC or Maximize clicks with **$2–4** cap to start.  
- **Expectation:** ~40–80 clicks/mo — learn winning theme, not fill calendar.

### Friday ads ritual (15 min)

1. Apply kill rule  
2. Note top LinkedIn creatives / Google search terms  
3. Move ≤$50 between LinkedIn ↔ Google only if one channel is clearly dead  
4. Log leads into Notion pipeline (same stages as GTM)

## 4. Website, measurement, rhythm

### Website wiring

| Item | Spec |
|------|------|
| Domain | https://sovereignwarden.com |
| Phase 1 CTA | Prefer sovereignty brief (email) for ads; site may keep discovery CTA treated as low-intent |
| Phase 2 CTA | Book discovery call → `#contact` / calendar (already on site) |
| Tracking | GA4 (or Plausible) + LinkedIn Insight Tag + Google Ads tag; conversion = form/booking |
| Phase switch | Adneo signed + one usable proof point → flip company CTA, ads, and primary site CTA |

### Five KPIs only

| KPI | 90-day target |
|-----|----------------|
| Company posts shipped | ≥3/week average |
| Paid learning | ≥1 clear winner theme (LI audience or Google ad group) |
| Meaningful actions | 8–20 total |
| Fit calls (phase 2) | 2–4 marketing-assisted |
| Cost per meaningful action | Track; don’t obsess until n≥10 |

Ignore follower vanity and impression-only metrics for decisions.

### Weekly founder rhythm

| Block | Work |
|-------|------|
| Mon ~30 min | Batch-write/schedule 3 company posts |
| Daily 10–15 min | Amplify + reply |
| Fri 15 min | Ads kill rule + KPI note |
| Monthly 45 min | Phase check, creative refresh, ≤$50 budget tilt |

- [ ] Sovereignty brief (1–2 page PDF) **or** confirm phase-1 ads use existing contact form

- [ ] LinkedIn: Audiences A + B; ~$10/day; 2–3 creatives live  
- [ ] Google: 2–3 search ad groups; negatives; ~$6.50/day; RSAs live

- [ ] Same ICP language as market/ICP docs  
- [ ] No free law pilots in ads or posts  
- [ ] Phase 2 flip when Adneo proof ready  
- [ ] Association channels deferred to Q3–Q4 2026  

### F. Later (after system runs)

- [ ] Sales advisor / warm ICP intros  
- [ ] Cyber MSP co-marketing post–case study  
- [ ] Raise ad budget only after a proven winner
