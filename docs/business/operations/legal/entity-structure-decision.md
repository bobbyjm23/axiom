## 2. Options evaluated

### Option A: NZ company only (rejected)

Operate Sovereign Warden through the existing NZ trading company.

| Pros | Cons |
|------|------|
| No new registration cost | AU buyers expect local contracting entity |
| Entity already exists | Cross-border Privacy Act / DPA complexity |
| | GST and tax friction on AU revenue |
| | Adneo procurement prefers AU vendor |
| | AU angels invest into AU Pty Ltd |
| | IP chain risk if NZ co ever held SW assets |

**Verdict:** Rejected. Does not fit AU-primary market or investor expectations.

### Option B: AU Pty Ltd as operating entity (selected)

Register **Sovereign Warden Pty Ltd** in Australia. Retain NZ trading co for unrelated activities only.

| Pros | Cons |
|------|------|
| Native fit for AU mid-market procurement | Registration cost (~$500–1,000) |
| Direct Privacy Act / DPA alignment | Requires AU resident director (founder is NZ-based) |
| Cleaner Adneo vendor relationship | Ongoing resident director arrangement |
| Standard for AU angel investors | |
| Clean IP chain: founder personal → AU Pty Ltd | |
| NSW governing law (consistent with existing NDA) | |

**Verdict:** Selected.

### Option C: Dual structure with IP in NZ (rejected)

NZ company holds IP; AU Pty Ltd licenses or contracts.

| Pros | Cons |
|------|------|
| Uses existing NZ entity | Unnecessary complexity for solo founder |
| | Investors prefer clean AU cap table and IP ownership |
| | Two entities to maintain |
| | Transfer pricing / tax advice needed |

**Verdict:** Rejected. Adds complexity without benefit at this stage.

## 3. Locked structure

```
Founder (NZ resident, 100% shareholder)
    │
    ├── NZ Trading Co ──► Other founder activities ONLY (no SW IP)
    │
    └── Sovereign Warden Pty Ltd (AU)
            ├── All SW IP (via assignment deed)
            ├── All customer contracts
            ├── All investor instruments (SAFE)
            └── AU Resident Director (s201A compliance seat)
```

### Key rules

1. **No Sovereign Warden IP** is assigned to the NZ trading company.
2. IP assignment path: **founder personal → AU Pty Ltd** via deed within 2 weeks of registration.
3. Founder holds **100% of shares** at formation; angel SAFE dilution follows bridge raise.
4. At least **one director must ordinarily reside in Australia** (s201A Corporations Act 2001).
5. Founder may serve as **non-resident director** in addition to the resident director.
